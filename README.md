# Gridrunner Academy

Gridrunner Academy is a cyberpunk-inspired STEAM platform teaching cybersecurity, networking, RF, and defensive hacking through safe hands-on hardware labs using ESP32s, Raspberry Pi, and portable field-tech kits.

## Projects

- Signal Siege
- Packet Quest
- RF Ops
- Mesh Raiders
